# Python built in generators

controls_if: if ... elif ... else: statements.
controls_whileUntil: while loops with while or until conditions.
controls_for: for loops with a numerical range.
controls_forEach: for loops iterating over a list.
controls_flow_statements: break and continue statements within loops.
controls_return: return statement.
Logic Blocks:

logic_compare: Comparison operators (==, !=, <, >, <=, >=).
logic_operation: Logical operators (and, or).
logic_negate: Logical negation (not).
logic_boolean: True and False boolean values.
logic_null: None value.
logic_ternary: Conditional expression (value_if_true if condition else value_if_false).
Math Blocks:

math_number: Numerical literals.
math_arithmetic: Arithmetic operators (+, -, *, /, **, %).
math_single: Mathematical functions like abs, sqrt, sin, cos, etc.
math_trig: Trigonometric functions (sin, cos, tan, asin, acos, atan).
math_constant: Mathematical constants like pi, e, Infinity.
math_number_property: Checks for number properties like even, odd, prime, positive, negative, divisible by.
math_round: Rounding functions (round, floor, ceil).
math_modulo: Modulo operator (%).
math_random_int: Generates a random integer within a range.
math_random_float: Generates a random floating-point number between 0 and 1.
math_atan2: Calculates the arctangent of y/x.
math_pow: Exponentiation (**).
Text Blocks:

text: String literals.
text_join: String concatenation using the + operator or "".join().
text_append: Appending text to a variable using +=.
text_length: Getting the length of a string using len().
text_isEmpty: Checking if a string is empty.
text_indexOf: Finding the first or last occurrence of a substring using find() or rfind().
text_charAt: Getting a character at a specific index using string indexing.
text_getSubstring: Extracting a substring using slicing.
text_changeCase: Changing the case of a string (upper(), lower(), title()).
text_trim: Removing leading and trailing whitespace using strip().
text_print: Using the print() function.
text_prompt_ext: Using the input() function for prompts.
List Blocks:

lists_create_empty: Creating an empty list ([]).
lists_create_with: Creating a list with specified items.
lists_repeat: Creating a list with a repeated item.
lists_length: Getting the length of a list using len().
lists_isEmpty: Checking if a list is empty.
lists_indexOf: Finding the first or last occurrence of an item in a list using index() or rindex().
lists_getIndex: Getting an item from a list at a specific index.
lists_setIndex: Setting an item in a list at a specific index.
lists_getSublist: Getting a sublist using slicing.
lists_sort: Sorting a list using sorted().
lists_split: Splitting a string into a list or joining a list into a string using split() and "".join().
Colour Blocks:

colour_picker: Generates a hexadecimal colour string.
colour_random: Generates a random hexadecimal colour string.
colour_rgb: Creates a colour from red, green, and blue components.
colour_blend: Blends two colours together.
Variable Blocks:

variables_get: Generates the variable name.
variables_set: Generates an assignment statement (variable = value).
Function Blocks:

procedures_defnoreturn: Defines a function without a return value using def.
procedures_defreturn: Defines a function with a return value using def and return.
procedures_callnoreturn: Calls a function without a return value.
procedures_callreturn: Calls a function with a return value.
